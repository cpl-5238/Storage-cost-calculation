### 1. Storage vs Demurrage Cost Comparison Tool
**File**: `compare_storage_vs_demurrage.py`

An interactive Streamlit application that compares warehouse storage charges and demurrage charges based on key dates and container specifications.

#### Features:
- Support for multiple destination countries (France, United Kingdom)
- Multiple container types (20'dc, 40'dc, 40'hc)
- Tiered pricing based on duration
- Detailed cost breakdown
- Interactive date selection

#### How to Run:
```bash
streamlit run compare_storage_vs_demurrage.py
```

Then open your browser to `http://localhost:8501`
